# Signal Messenger Addon

Home Assistant addon voor [Signal Messenger](https://signal.org/) via een REST API.

Gebaseerd op [signal-cli-rest-api](https://github.com/bbernhard/signal-cli-rest-api) door **[@bbernhard](https://github.com/bbernhard)** — alle credits gaan naar hem.

## Features

- Stuur en ontvang Signal berichten vanuit HA automations
- Registreer een nieuw nummer of koppel als secundair apparaat via QR-code
- Meerdere uitvoermodi (normal, native, json-rpc)
- REST API op poort 8080

Zie [DOCS.md](DOCS.md) voor installatie en configuratie.
