# Mealie Addon

Home Assistant addon voor [Mealie](https://mealie.io/) — zelf-gehoste receptenmanager met meal planning en boodschappenlijst.

## Features

- Recepten importeren via URL
- Meal planning
- Boodschappenlijst
- Data opgeslagen in `/share/mealie/`
- Bereikbaar via HA Ingress of poort 9000

Zie [DOCS.md](DOCS.md) voor configuratie.
